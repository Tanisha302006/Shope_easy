# ShopEasy

A full-stack e-commerce project built with React, Tailwind CSS, Node.js, Express, and MongoDB.

## Project Structure

### Backend (`/backend`)
- **models/**: Data schemas for MongoDB.
- **routes/**: API route definitions.
- **controllers/**: Business logic for routes.
- **middleware/**: Custom Express middleware (Auth, Error handling).
- **config/**: Database and global configurations.
- **utils/**: Shared utility functions.

### Frontend (`/frontend`)
- **pages/**: View components.
- **components/**: Reusable UI widgets.
- **services/**: API integration logic.
- **hooks/**: Custom React hooks.
- **context/**: State management logic.
- **assets/**: Static assets.

## Getting Started

1. Clone the project.
2. Run `npm run install-all` in the root directory.
3. Configure your `.env` file in the root.
4. Run `npm run dev` to start both servers.
